# Manual Test Case

## Title
Main page should display navigation buttons: Docs, API, Community

## Preconditions
- Browser is available
- Internet access to `https://playwright.dev/`

## Steps
1. Open `https://playwright.dev/`.
2. Observe the main navigation area in the page header.
3. Verify that the `Docs` navigation link is present and visible.
4. Verify that the `API` navigation link is present and visible.
5. Verify that the `Community` navigation link is present and visible.
6. Click `Docs` and verify the page opens the documentation section.
7. Return to the main page.
8. Click `API` and verify the page opens the API documentation section.
9. Return to the main page.
10. Click `Community` and verify the link opens the community destination.

## Expected Result
- The main page displays `Docs`, `API`, and `Community`
- Each item is visible and accessible as a navigation link
- Each link opens the correct destination
